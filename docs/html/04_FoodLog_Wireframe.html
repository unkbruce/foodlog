FoodLog 와이어프레임

FoodLog 웹 서비스의 주요 화면 구조와 화면 이동 흐름을 모바일 우선 관점에서 와이어프레임 형태로 정리한 문서입니다.

프로젝트명FoodLog
화면 기준모바일 화면
주요 화면목록, 추가, 상세, 피드백

PDF 저장 시 브라우저 인쇄 설정에서 머리글과 바닥글을 끄면 문서 본문만 깔끔하게 출력됩니다.

핵심 요약 FoodLog는 PC와 모바일 브라우저에서 사용할 수 있는 웹 서비스이며, 초기 화면 설계는 모바일 사용성을 우선하여 구성한다. 화면 구조는 음식 기록 목록을 중심으로 음식 추가, 상세/수정, 식단 피드백 화면으로 이동하는 단순한 구조이다.

화면 목적

구성 요소설명
